Our Team

Components

Music file transfer

Design this application with P2P architecture. Users in the same group can get any song in the group play list that is closer to the user’s location.

Server and Client

Client sends requests to server. Server take care of all the requests.

Playlist consistency

Users can see synchronized play list from their own view, and add or delete songs in the list are supported.

Fault tolerance

When one node died in the middle of transfer a file, other node who has this file can keep sending this file to make sure this process won’t fail.
